gbaemu4DS by ichfly

this is the gba emulator for the DS

This is a gba emulator for DS(i) work in process

Install

1.patch your gba files 2.copy the patched gba files to your flashcard 3.dldi patch the nds file for your flashcard (most flashcard do this on the fly so you may don't need that step) 4.copy the nds file to your flashcard 5.Start the nds file 6.select fat:/ 7.select a gba file 8.set the malloc size that work (the size that is there at the start works normaly) 9. press start 10.the gba game should start
